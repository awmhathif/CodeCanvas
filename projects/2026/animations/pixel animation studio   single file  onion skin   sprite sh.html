Export PNG / Sheet / JSON

Export PNG

Export current frame or full animation as a sprite sheet.

Sprite sheet order is frame 1..N left-to-right, top-to-bottom. Nearest-neighbor scaling.

Export Meta (JSON)

Generate a simple JSON descriptor for your sprite sheet.

Includes frame size, count, fps, and frame rectangles (x,y,w,h).

Import

Import a PNG sprite sheet or a single frame PNG (same size recommended).

For sheet import: uses current canvas frame size as tile size. Extra pixels are ignored.

Project

Project JSON includes all frames (compressed as base64 PNG per frame). Stored in localStorage too.
