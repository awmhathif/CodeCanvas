⚠️ IDB Honest Caveat
IndexedDB is exported on first run as a snapshot. It is NOT replaced with file-backed storage. Apps that actively write to IndexedDB after launch still use Chromium's internal store. The export is a migration snapshot only.
🗄 SQLite Notes-App Fit
SQLite with WAL mode is ideal for notes apps: crash-safe, single .db file, easy to backup, supports complex queries if you add them later. The wrapper exposes the same simple key/value API — you don't need to write SQL.
